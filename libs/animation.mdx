## Most popular

<CardGroup cols={3}>
  <Card title="lottie-ios" icon="film" href="https://github.com/airbnb/lottie-ios">
    An iOS library by Airbnb to natively render After Effects vector animations exported as JSON with Bodymovin.
  </Card>
  <Card title="Spring" icon="wand-magic-sparkles" href="https://github.com/MengTo/Spring">
    A library to simplify iOS animations with a simple, expressive API. One of the most starred Swift animation repos.
  </Card>
  <Card title="IBAnimatable" icon="paintbrush" href="https://github.com/IBAnimatable/IBAnimatable">
    Design and prototype UI, interaction, navigation, transition and animation for App Store ready apps in Interface Builder.
  </Card>
</CardGroup>

## All animation libraries

<AccordionGroup>
  <Accordion title="General-purpose frameworks">
    Full-featured animation frameworks that cover a wide range of use cases.

    | Library | Description |
    |---------|-------------|
    | [Advance](https://github.com/timdonnelly/Advance) | A powerful animation framework for iOS, tvOS, and OS X. |
    | [EasyAnimation](https://github.com/icanzilb/EasyAnimation) | Takes the power of `UIView.animateWithDuration` to a whole new level. |
    | [FlightAnimator](https://github.com/AntonTheDev/FlightAnimator) | Natural blocks-based Core Animation framework. |
    | [IBAnimatable](https://github.com/IBAnimatable/IBAnimatable) | Design and prototype UI, interaction, navigation, transition and animation in Interface Builder. |
    | [Sica](https://github.com/cats-oss/Sica) | Simple Interface Core Animation. Run type-safe animations sequentially or in parallel. |
    | [Spring](https://github.com/MengTo/Spring) | A library to simplify iOS animations. |
    | [spruce-ios](https://github.com/willowtreeapps/spruce-ios) | Choreograph animations on screen. |
    | [TheAnimation](https://github.com/marty-suzuki/TheAnimation) | Type-safe `CAAnimation` wrapper. Prevents setting wrong type values. |
    | [ViewAnimator](https://github.com/marcosgriselli/ViewAnimator) | Brings your UI to life with just one line. |
    | [YapAnimator](https://github.com/yapstudios/YapAnimator) | Fast and friendly physics-based animation system. |
  </Accordion>

  <Accordion title="Physics & spring animations">
    Libraries based on physical simulation and spring dynamics.

    | Library | Description |
    |---------|-------------|
    | [CocoaSprings](https://github.com/MacPaw/CocoaSprings) | Interactive spring animations for iOS/macOS. |
    | [Ease](https://github.com/roberthein/Ease) | Animate everything with Ease. |
    | [Interpolate](https://github.com/marmelroy/Interpolate) | Interpolation framework for creating interactive gesture-driven animations. |
    | [Stellar](https://github.com/AugustRush/Stellar) | A physical animation library. |
    | [SpriteKitEasingSwift](https://github.com/craiggrummitt/SpriteKitEasingSwift) | Better easing for SpriteKit. |
  </Accordion>

  <Accordion title="Gradient & color animations">
    Libraries for animating gradients and color transitions.

    | Library | Description |
    |---------|-------------|
    | [AnimatedGradient](https://github.com/exyte/AnimatedGradient) | Animated linear gradient library written with SwiftUI. |
    | [Pastel](https://github.com/cruisediary/Pastel) | Gradient animation effect like Instagram. |
  </Accordion>

  <Accordion title="Particle & visual effects">
    Libraries for particle systems, pulses, and visual flair.

    | Library | Description |
    |---------|-------------|
    | [Comets](https://github.com/cruisediary/Comets) | Animating particles. |
    | [Pulsator](https://github.com/shu223/pulsator) | Pulse animation for iOS. |
  </Accordion>

  <Accordion title="Scroll & collection animations">
    Libraries that animate views driven by scroll position or collection view interactions.

    | Library | Description |
    |---------|-------------|
    | [ChainPageCollectionView](https://github.com/jindulys/ChainPageCollectionView) | Fancy two-level collection view layout and animation. |
    | [Gemini](https://github.com/shoheiyokoyama/Gemini) | Rich scroll-based animation framework. |
  </Accordion>

  <Accordion title="SVG & vector animations">
    Libraries for animating vector and SVG content.

    | Library | Description |
    |---------|-------------|
    | [Elephant](https://github.com/s2mr/Elephant) | Elegant SVG animation kit. |
    | [lottie-ios](https://github.com/airbnb/lottie-ios) | An iOS library to natively render After Effects vector animations. |
  </Accordion>

  <Accordion title="Presentation & onboarding animations">
    Libraries for animated tutorials, release notes, and guided pages.

    | Library | Description |

    | [Presentation](https://github.com/hyperoslo/Presentation) | A library to help you make tutorials, release notes, and animated pages. |
  </Accordion>
