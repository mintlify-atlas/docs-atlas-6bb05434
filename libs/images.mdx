</Note>

## Most popular

<CardGroup cols={2}>
  <Card title="Kingfisher" icon="crown" href="https://github.com/onevcat/Kingfisher">
    A powerful, pure-Swift library for downloading and caching images from the web. Highly optimized with a clean API — the most popular Swift image library.
  </Card>
  <Card title="Nuke" icon="bolt" href="https://github.com/kean/Nuke">
    Advanced framework for loading, caching, processing, displaying, and preheating images. Highly configurable pipeline architecture.
  </Card>
  <Card title="AlamofireImage" icon="network-wired" href="https://github.com/Alamofire/AlamofireImage">
    An image component library for Alamofire. Provides image downloading, caching, and filtering with Alamofire's networking layer.
  </Card>
  <Card title="SwiftyGif" icon="image" href="https://github.com/alexiscreuzot/SwiftyGif">
    High performance GIF engine for iOS.
  </Card>
</CardGroup>

## All image libraries

<AccordionGroup>
  <Accordion title="Image loading & caching">
    Libraries for downloading images from the network and caching them efficiently.

    | Library | Description |
    |---------|-------------|
    | [AlamofireImage](https://github.com/Alamofire/AlamofireImage) | Image component library for Alamofire. |
    | [HanekeSwift](https://github.com/Haneke/HanekeSwift) | A lightweight generic cache for iOS with extra love for images. |
    | [ImageLoader](https://github.com/hirohisa/ImageLoaderSwift) | A lightweight and fast image loader for iOS. |
    | [Kingfisher](https://github.com/onevcat/Kingfisher) | Image download and caching. |
    | [MapleBacon](https://github.com/JanGorman/MapleBacon) | Image download and caching library. |
    | [Moa](https://github.com/evgenyneu/moa) | An image download extension of the image view for iOS, tvOS, and macOS. |
    | [Nuke](https://github.com/kean/Nuke) | Advanced framework for loading, caching, processing, displaying, and preheating images. |
    | [SwiftWebImage](https://github.com/HotWordland/SwiftWebImage) | SwiftUI image downloader with performant LRU mem/disk cache. |
  </Accordion>

  <Accordion title="Image viewers & browsers">
    Libraries for photo viewers, galleries, and lightbox-style displays.

    | Library | Description |
    |---------|-------------|
    | [Agrume](https://github.com/JanGorman/Agrume) | A lemony fresh iOS image viewer. |
    | [ATGMediaBrowser](https://github.com/altayer-digital/ATGMediaBrowser) | Image slide-show viewer with multiple predefined transition styles. |
    | [AXPhotoViewer](https://github.com/alexhillc/AXPhotoViewer) | An iPhone/iPad photo gallery viewer for viewing large or small numbers of photos. |
    | [DTPhotoViewerController](https://github.com/tungvoduc/DTPhotoViewerController) | A fully customizable photo viewer inspired by Facebook photo viewer. |
    | [ImageViewer](https://github.com/Krisiacik/ImageViewer) | An image viewer à la Twitter. |
    | [Lightbox](https://github.com/hyperoslo/Lightbox) | A convenient and easy to use image viewer for your iOS app. |
    | [SKPhotoBrowser](https://github.com/suzuki-0000/SKPhotoBrowser) | Simple PhotoBrowser/Viewer inspired by Facebook and Twitter photo browsers. |
  </Accordion>

  <Accordion title="Image pickers">
    Libraries for selecting images from the photo library or camera.

    | Library | Description |
    |---------|-------------|
    | [FacebookImagePicker](https://github.com/floriangbh/FacebookImagePicker) | Facebook album photo picker. |
    | [FMPhotoPicker](https://github.com/congnd/FMPhotoPicker) | A modern, simple, zero-dependency photo picker with an elegant and customizable image editor. |
    | [YPImagePicker](https://github.com/Yummypets/YPImagePicker) | Instagram-like image picker & filters for iOS. |
  </Accordion>

  <Accordion title="Image processing & editing">
    Libraries for filtering, cropping, masking, and GPU-accelerated image processing.

    | Library | Description |
    |---------|-------------|
    | [Brightroom](https://github.com/FluidGroup/Brightroom) | An image editor and engine using CoreImage. |
    | [FaceCrop](https://github.com/Ancestry/FaceCrop) | Detect and center faces in your images using Apple's Vision Framework. |
    | [FlexibleImage](https://github.com/kawoou/FlexibleImage) | A simple way to play with images. |
    | [GPUImage 2](https://github.com/BradLarson/GPUImage2) | A BSD-licensed framework for GPU-accelerated video and image processing. |
    | [GPUImage 3](https://github.com/BradLarson/GPUImage3) | GPU-accelerated video and image processing using Metal. |
    | [Harbeth](https://github.com/yangKJ/Harbeth) | Metal API for GPU-accelerated graphics, video, and camera filter framework. |
    | [ImageDetect](https://github.com/Feghal/ImageDetect) | Detect and crop faces, barcodes, and texts in images with iOS 11 Vision API. |
    | [Kanvas](https://github.com/tumblr/kanvas-ios) | An iOS library for adding effects, drawings, text, stickers, and making GIFs. |
    | [MCScratchImageView](https://github.com/JaylenCoding/MCScratchImageView) | A custom `ImageView` used to cover the surface of other views like a scratch card. |
    | [Sharaku](https://github.com/makomori/Sharaku) | Image filtering UI library like Instagram. |
    | [TinyCrayon](https://github.com/TinyCrayon/TinyCrayon-iOS-SDK) | A smart and easy-to-use image masking and cutout SDK for mobile apps. |
    | [Toucan](https://github.com/gavinbunney/Toucan) | Image processing API. |
    | [ZImageCropper](https://github.com/ZaidPathan/ZImageCropper) | Crop image in any shape. |
  </Accordion>

  <Accordion title="SVG & vector images">
    Libraries for rendering and working with SVG and vector formats.

    | Library | Description |
    |---------|-------------|
    | [Rough](https://github.com/bakhtiyork/Rough) | Draw in a sketchy, hand-drawn-like style. |
    | [Snowflake](https://github.com/onmyway133/Snowflake) | Work with SVG. |
    | [SwiftDraw](https://github.com/swhitty/SwiftDraw) | Converts SVG images to `UIImage`, `NSImage`, and generates CoreGraphics source code. |
    | [SwiftSVG](https://github.com/mchoe/SwiftSVG) | A single-pass SVG parser with multiple interface options (String, `NS`/`UIBezierPath`, `CAShapeLayer`, and `NS`/`UIView`). |
  </Accordion>

  <Accordion title="GIF & animated images">
    Libraries for displaying and working with animated GIFs and APNG.

    | Library | Description |
    |---------|-------------|
    | [APNGKit](https://github.com/onevcat/APNGKit) | High performance and delightful way to play with APNG format in iOS. |
    | [gifu](https://github.com/kaishin/gifu) | Highly performant animated GIF support for iOS. |
    | [SwiftyGif](https://github.com/alexiscreuzot/SwiftyGif) | High performance GIF engine. |
  </Accordion>

  <Accordion title="Avatars & identicons">
    Libraries for generating avatars and profile images programmatically.

    | Library | Description |
    |---------|-------------|
    | [BlockiesSwift](https://github.com/Boilertalk/BlockiesSwift) | Unique blocky identicons/profile picture generator. |
    | [LetterAvatarKit](https://github.com/vpeschenkov/LetterAvatarKit) | A `UIImage` extension that generates letter-based avatars. |
  </Accordion>

  <Accordion title="Panorama & specialized viewers">
    Libraries for panoramas, passports, and specialized image display.

    | Library | Description |
    |---------|-------------|
    | [CTPanoramaView](https://github.com/scihant/CTPanoramaView) | Displays spherical or cylindrical panoramas with touch or motion controls. |
    | [PassportScanner](https://github.com/evermeer/PassportScanner) | Scan the MRZ code of a passport and extract name, number, nationality, dates, and personal number. |
  </Accordion>

  <Accordion title="Image utilities">
    Miscellaneous utilities for image URLs, color extraction, stickers, and more.

    | Library | Description |
    |---------|-------------|
    | [ImageScout](https://github.com/kaishin/ImageScout) | Implementation of fastimage — supports PNG, GIF, and JPEG. |
    | [ImgixSwift](https://github.com/imgix/imgix-swift) | Easily update image URLs to be fast and responsive. |
    | [JLStickerTextView](https://github.com/Textcat/JLStickerTextView) | A `UIImageView` that allows adding multiple labels, editable and renderable as image. |
    | [SwiftGen-Assets](https://github.com/SwiftGen/SwiftGen#assets-catalogs) | Auto-generate `enums` for all your `UIImages` from Asset Catalogs. |
    | [UIImageColors](https://github.com/jathu/UIImageColors) | iTunes style color fetcher for `UIImage`. |
  </Accordion>
</AccordionGroup>
