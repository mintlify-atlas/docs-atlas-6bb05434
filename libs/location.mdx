## Location

Libraries for requesting location permissions, accessing GPS data, and monitoring location changes.

<CardGroup cols={2}>
  <Card title="AsyncLocationKit" href="https://github.com/AsyncSwift/AsyncLocationKit" icon="location-dot">
    Wrapper for Apple's CoreLocation framework using modern Swift concurrency (async/await).
  </Card>
  <Card title="STLocationRequest" href="https://github.com/SvenTiigi/STLocationRequest" icon="location-dot">
    Elegant and simple 3D flyover location permission request screen.
  </Card>
  <Card title="SwiftLocation" href="https://github.com/malcommac/SwiftLocation" icon="location-dot">
    Location and beacon monitoring library with support for iBeacon ranging and geofencing.
  </Card>
  <Card title="LocoKit" href="https://github.com/sobri909/LocoKit" icon="location-dot">
    Location and activity recording framework for iOS, including timeline segmentation.
  </Card>
</CardGroup>

<Tip>SwiftLocation also covers iBeacon monitoring — see the Hardware page for more hardware-oriented location uses.</Tip>

## Maps

Libraries for working with MKMapView, clustering annotations, and building map-based features.

<CardGroup cols={2}>
  <Card title="Cluster" href="https://github.com/efremidze/Cluster" icon="map">
    Easy map annotation clustering for MKMapView with customizable cluster views.
  </Card>
  <Card title="FlyoverKit" href="https://github.com/SvenTiigi/FlyoverKit" icon="map">
    Present stunning 360° flyover views on MKMapView with zero effort and full configuration.
  </Card>
  <Card title="GEOSwift" href="https://github.com/GEOSwift/GEOSwift" icon="map">
    Work with geographic models and calculate intersections, overlaps, and projections.
  </Card>
  <Card title="LocoKit" href="https://github.com/sobri909/LocoKit" icon="map">
    Location and activity recording framework — also suitable for timeline-based map features.
  </Card>
</CardGroup>
