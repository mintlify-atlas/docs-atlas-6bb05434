These resources come directly from Apple and the Swift open source project. Whether you are just getting started or need a precise reference for API design, these are the authoritative sources.

<CardGroup cols={2}>
  <Card
    title="API Design Guidelines"
    icon="book-open"
    href="https://www.swift.org/documentation/api-design-guidelines/"
  >
    The canonical reference for naming conventions and design principles in Swift APIs. Read this to understand how Apple structures its own frameworks and how you should structure yours.
  </Card>
  <Card
    title="The Swift Programming Language (Apple eBook)"
    icon="book"
    href="https://books.apple.com/us/book/the-swift-programming-language-swift-5-7/id881256329"
  >
    Apple's official eBook covering Swift from the ground up. It walks you through syntax, types, control flow, and advanced features — an essential starting point for beginners and a reliable reference for experienced developers.
  </Card>
  <Card
    title="Getting Started"
    icon="rocket"
    href="https://www.swift.org/getting-started/"
  >
    The official Swift.org getting-started guide. It covers how to install Swift, run your first program, and find your way around the toolchain. Start here if you are new to the language or setting up a new environment.
  </Card>
  <Card
    title="Introducing SwiftUI"
    icon="display"
    href="https://developer.apple.com/tutorials/swiftui"
  >
    Apple's interactive tutorial series for SwiftUI with over four hours of hands-on content. You build real apps step by step, learning layout, data flow, and platform integration as you go.
  </Card>
</CardGroup>
