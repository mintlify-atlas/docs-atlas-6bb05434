Awesome Swift is a community-maintained, curated list of Swift libraries, frameworks, tools, and resources. It covers iOS, macOS, tvOS, watchOS, and Linux development. The list currently tracks **1,098 projects** and is updated regularly — last updated March 28, 2026.

The project is part of the [awesome](https://github.com/sindresorhus/awesome) ecosystem: a collection of high-quality, curated lists across the open source world.

## Linux compatibility

Libraries marked with a 🐧 badge are compatible with Linux via Swift on Linux and Swift Package Manager. If you're building server-side or cross-platform Swift, filter for this badge to find libraries that work outside of Apple platforms.

## How the list is structured

The collection is organized into three top-level areas:

### Guides

The Guides section contains learning resources:

- **Newsletter** — the [Open Source Updates for Swift Projects](https://ossp-updates.beehiiv.com/) bi-weekly newsletter
- **Official Guides** — Apple's Swift documentation, the Swift Programming Language eBook, and the official getting started guide
- **Style Guides** — coding style references from Airbnb, Google, LinkedIn, and Raywenderlich
- **Third-party Guides** — community tutorials, playgrounds, and video courses

### Tools

The Tools section covers the development toolchain:

- **Dependency managers** — CocoaPods, Carthage, Swift Package Manager, Mint, Accio, and Swiftly
- **Editor support** — Emacs, Vim, and Google Colaboratory integrations
- **Boilerplates** — project templates and module starters
- **REPL** — online playgrounds like SwiftFiddle and Online Swift Playground
- **Misc** — code generators, Xcode utilities, and build tools like Tuist and XcodeGen

### Libs

The Libs section is the largest part of the collection. It is organized by domain, covering over 60 categories including:

- UI components, animation, layout, and image handling
- Networking, caching, and data management (Core Data, Realm, SQLite, JSON)
- App architecture patterns (MVVM, VIPER, Composable Architecture, Redux)
- Hardware access (Bluetooth, camera, sensors, ARKit)
- Security, cryptography, and keychain access
- AI, machine learning, and CoreML
- Testing, logging, and analytics

## How to navigate this site

<Steps>
  <Step title="Use the sidebar tabs">
    The site has two main tabs: **Overview** (guides, tools, and this introduction) and **Libraries** (all lib categories grouped by domain).
  </Step>
  <Step title="Browse by category">
    Each sidebar group contains related categories. Open a group to see all libraries in that area, with links to GitHub and short descriptions.
  </Step>
  <Step title="Check the Linux badge">
    If you need Linux compatibility, look for the 🐧 badge next to a library's name. Only entries with this tag are confirmed to work on Linux.
  </Step>
  <Step title="Go to the source">
    The list is generated from [`contents.json`](https://github.com/matteocrippa/awesome-swift/blob/master/contents.json) in the GitHub repository. Every entry links directly to its project page.
  </Step>
</Steps>

<Note>
  The README is auto-generated from `contents.json`. If you want to suggest a new library or fix an entry, edit the JSON file — not the README. See the [contributing guide](/contributing) for the full process.
</Note>

## What qualifies for the list

Not every Swift library appears here. To be included, a project must meet these criteria:

- Actively maintained and used by the community
- Well documented with a README in English
- At least 15 GitHub stars
- Supports Swift 5 or later
- Released under Apache 2.0, BSD 2/3, GPL, LGPL, or MIT license
- Compatible with the latest SDK

<Tip>
  If you find a library that no longer meets these criteria, open an issue on [GitHub](https://github.com/matteocrippa/awesome-swift/issues) to have it reviewed for removal.
</Tip>
