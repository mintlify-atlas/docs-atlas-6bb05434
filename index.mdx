<Info>
  Libraries marked with 🐧 are compatible with Linux via Swift on Linux / Swift Package Manager.
</Info>

## How to use this list

<Steps>
  <Step title="Find your category">
    Browse the **Libraries** tab in the sidebar to find the category that matches what you need — UI, Networking, Security, and more.
  </Step>
  <Step title="Pick a library">
    Each entry includes a link to the library's GitHub repository and a brief description. Look for the 🐧 badge if Linux compatibility matters to you.
  </Step>
  <Step title="Add it to your project">
    Most libraries support Swift Package Manager. Add them via Xcode's package manager or in your `Package.swift` file.
  </Step>
  <Step title="Contribute back">
    Found a great library that's missing? See the [contributing guide](/contributing) to submit a pull request.
  </Step>
</Steps>

<Tip>
  This list is generated from [contents.json](https://github.com/matteocrippa/awesome-swift/blob/master/contents.json) — a structured data file that powers the README. If you want to contribute, edit that file rather than the README directly.
</Tip>
